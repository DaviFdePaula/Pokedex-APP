## :dart: About ##

Um catalogo pokemon, Pokedex, feito com React JS. Consumindo a Pokeapi e usando localstorage.

## :rocket: Technologies ##

The following tools were used in this project:
- [React](https://pt-br.reactjs.org/)
- [Poke Api](https://pokeapi.co/)

## :white_check_mark: Requirements ##

Antes de iniciar vocÃª precisa ter [Git](https://git-scm.com), [Node](https://nodejs.org/en/) e NPM(https://www.npmjs.com/) instalados.

## :checkered_flag: Starting ##

```bash
# Clone this project
$ git clone https://github.com/DaviFdePaula/pokedex

# Access
$ cd pokedex

# Install dependencies
$ npm install 

# Run the project
$ npm start

# The server will initialize in the <http://localhost:3000>
```
